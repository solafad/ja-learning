### Part A
1. 宿 — しゅく — Inn / house
2. 題 — だい — Topic / subject
3. 建 — たてる — To build
4. 寺 — てら / じ — Buddhist temple
5. 絵 — え — Picture / painting

Compounds: 宿題（しゅくだい）, 問題（もんだい）, 建物（たてもの）, 絵文字（えもじ）, 清水寺（きよみずでら）

### Part B
1. 猫 — ねこ — Cat
2. 馬 — うま — Horse
3. 鳥 — とり — Bird
4. 牛 — うし / ぎゅう — Cattle
5. 魚 — さかな / ぎょ — Fish

Compounds: 牛肉（beef）, 牛乳（milk）, 和牛（wagyu Japanese beef）, 金魚（goldfish）, 人魚（mermaid）

### Part C
1. 発（はつ）— 発明する、出発する
2. 付（つき）— 付き合う、日付
3. 欲（ほし/よく）— 欲しい、食欲
4. 集（あつめる）— 集める、集まる
5. 始（はじめる）— 始める、始まる
