1. 近 — ちか-い / きん — near / close

3. 可 — か — allowed
4. 暗 — くら-い — dark
5. 不 — ふ — negative prefix (un- / in-)

Compounds:
1. 近い（ちかい）— near
2. 近所（きんじょ）— neighbourhood

4. 不便（ふべん）— inconvenient
5. ペット可（ペットか）— pets allowed

7. 不安（ふあん）— insecurity / anxiety
8. 不幸（ふこう）— unhappiness
