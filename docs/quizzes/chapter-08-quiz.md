## Part E — だけ vs しか

Choose the correct word and complete the sentence.

| # | Sentence | だけ or しか? |
|---|----------|------------|
| 1 | 五百円\_\_\_ありません。*(I only have 500 yen — not enough)* | |
| 2 | 一時間\_\_\_ゲームをします。*(I play games for just an hour — I'm happy with that)* | |
| 3 | 日曜日\_\_\_ありません。*(The only day off is Sunday — unsatisfied)* | |
| 4 | 私\_\_\_です。*(It's only me — neutral)* | |

**Note:** しか → verb must be **negative**. だけ → verb stays **positive**.
