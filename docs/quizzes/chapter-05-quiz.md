### Part A
1. 押 — おす — To press / push
2. 伝 — つたえる — To tell / inform
3. 消 — きえる — (Lights/appliances) go out
4. 港 — こう — Port
5. 号 — ごう — Number

Compounds: 空港（くうこう）airport, 電話番号（でんわばんごう）telephone number, 手伝う（てつだう）to help, 伝える（つたえる）to tell

### Part B
1. 飛 — とぶ / ひ — To fly
2. 機 — き — Machine
3. 便 — びん — Flight / service
4. 席 — せき — Seat
5. 留 — りゅう — Stay (study abroad)

Compounds: 飛行機（ひこうき）plane, ゆう便きょく（ゆうびんきょく）post office, 座席（ざせき）seat, 留学生（りゅうがくせい）exchange student
