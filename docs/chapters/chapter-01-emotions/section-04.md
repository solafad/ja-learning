# Expressing happiness

## 1. Vocabulary List

| Japanese      | Romaji   | English                           |
|---------------|----------|-----------------------------------|
| 幸せ（しあわせ） | shiawase | Happy / Happiness (long-term)     |
| 今（いま）     | ima      | Now                               |
| 僕（ぼく）     | boku     | I / Me (masculine, polite/casual) |
| うれしい       | ureshii  | Glad / Happy (situational)        |
| おこる         | okoru    | To get angry                      |
| さびしい       | sabishii | Lonely                            |

---

## 2. Grammar & Sentences

**Expressing Current State:** Use the time marker 今 (いま) to describe a current emotional state.

- **Structure:** `[Subject]` は 今 `[Emotion]` です
- **Key Phrase:** 僕は今幸せです。*(Boku wa ima shiawase desu.)* — "I am happy now."

**Pronoun Usage:** 僕 (ぼく) is used to refer to oneself, common in masculine speech but widely understood.
