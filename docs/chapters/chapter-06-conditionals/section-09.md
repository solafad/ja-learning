## 2. Grammar & Sentences

### ば Form — Quick Review

**Structure:** Change the **-u** ending to the corresponding **-e** sound, then add **ば**.

| Dictionary | ば Form | English |
|------------|---------|---------|
| 食べる | 食べれば | if (I/you) eat |
| できる | できれば | if possible |
| ある | あれば | if there is |
| 行く | 行けば | if (I/you) go |

**Examples from the lesson:**
- おいしい食べ物を食べれば、元気になります。*(Oishii tabemono wo tabereba, genki ni narimasu.)* — "If I eat delicious food, I'll be better."
- できれば、さるを見に行きたいです。*(Dekireba, saru wo mi ni ikitai desu.)* — "If I can, I want to go and see monkeys."
- できれば、木曜日に行きたいのですが。*(Dekireba, mokuyoubi ni ikitai no desu ga.)* — "If possible, I would like to go on Thursday."

---

### Rule 1: ば Requires a Good Result or Speaker's Wish

A sentence using **〜ば** must express a **good result** or the **speaker's wishes**. If the result is negative or undesirable, use **〜たら** instead.

| | Japanese | English | Note |
|-|----------|---------|------|
| O | アイスクリームを食べたら、おなかがいたくなります。 | If I eat an ice cream, I'll have a stomach ache. | たら ✓ — bad result |
| X | ~~アイスクリームを食べれば、おなかがいたくなります。~~ | | ば ✗ — stomach ache is not a good result |
| O | アイスクリームを食べれば、うれしくなります。 | If I eat an ice cream, I'll be glad. | ば ✓ — good/wished result |
| O | 学校を休んだら、友だちに会えません。 | If I take a day off school, I can't see my friends. | たら ✓ — not seeing friends = unwanted |
| X | ~~学校を休めば、友だちに会えません。~~ | | ば ✗ — unnatural here |
| O | ビールを飲みすぎたら、あたまがいたくなりますよ。 | If you drink too much beer, you'll have a headache. | たら ✓ — headache is bad |
| X | ~~ビールを飲みすぎれば、あたまがいたくなりますよ。~~ | | ば ✗ — headache is not a good result |

---

### Rule 2: たら Can Mean "When" or "After" (Certainty)

When the たら clause describes something **certain to happen**, たら can mean **"when"** or **"after"** — not just "if."

| | Japanese | English | Note |
|-|----------|---------|------|
| ① | 田中さんが来たら、私は帰ります。 | **When** Tanaka-san comes, I'll go home. | Certainty — Tanaka-san will come |
| ② | 田中さんが来れば、私は帰ります。 | **If** Tanaka-san comes, I'll go home. | Uncertainty — unclear if coming |

**More examples of たら as "when":**
- 十二時になったら、帰ります。*(Juuniji ni nattara, kaerimasu.)* — "When it's 12 o'clock, I'll go home."
- 朝起きたら、シャワーをあびます。*(Asa okitara, shawaa wo abimasu.)* — "When I get up in the morning, I'll have a shower."

---

### Rule 3: ば Cannot Be Used for Past Events

**Verb 1 たら、Verb 2** — Verb 1 must happen first, and Verb 2 happens next. Both verbs **can describe past events**.  
This is **not** the case with **ば** — ば cannot be used when the result is a past event (〜ました).

| | Japanese | English | Note |
|-|----------|---------|------|
| O | 日本に行ったら、日本語が上手になりました。 | After I went to Japan, I became good at Japanese. | たら ✓ — past sequence |
| X | ~~日本に行けば、日本語が上手になりました。~~ | | ば ✗ — past event in result |
| O | 先週、カフェに行ったら、たけるさんに会いました。 | When I went to the cafe last week, I saw Takeru. | 会いました = past event → たら only |
| O | 薬を飲んだら、元気になりました。 | After I took some medicine, I felt better. | 元気になりました = past event → たら only |

---

### もし — Emphasising "If"

To **emphasise** the conditional "if", add **もし** at the beginning of ～たら or ～ば sentences.

**Structure:** `もし` + `[condition in ば or たら]` + `[result]`

| Japanese | Romaji | English |
|----------|--------|---------|
| もしお金があれば、留学したいです。 | Moshi okane ga areba, ryuugaku shitai desu. | If I have the money, I'd like to go and study abroad. |
| もし日本に行ったら、何をしますか。 | Moshi nihon ni ittara, nani wo shimasu ka. | If you go to Japan, what will you do? |
| もしたくさんお金があれば、幸せですか。 | Moshi takusan okane ga areba, shiawase desu ka. | If you have a lot of money, will you be happy? |
| もし時間があれば、手伝ってください。 | Moshi jikan ga areba, tetsudatte kudasai. | If you have time, please help me. |
| もしお金があれば、世界を旅行したいです。 | Moshi okane ga areba, sekai wo ryokou shitai desu. | If I have the money, I'd like to travel the world. |
| もしお金があったら、大きい家を建てたいです。 | Moshi okane ga attara, ookii ie wo tatai desu. | If I had the money, I'd like to build a big house. |

> **Note:** もし can be used with both ～ば and ～たら. It simply adds extra emphasis on the conditional "if."

---

### Summary: たら vs ば

| Rule | たら | ば |
|------|------|----|
| Good or wished result | ✓ | ✓ |
| Bad / undesired result | ✓ | ✗ |
| Certain event ("when") | ✓ | ✗ |
| Past event in result clause | ✓ | ✗ |
| Imaginary / unreal situations | ✓ | ✗ |
| With requests (〜てください) | ✓ | ✗ |

---

## 3. Learning Context

- **Grammar Focus:** Three key differences between たら and ば — result sentiment, certainty ("when"), and past events.
- **もし:** Optional particle that emphasises the "if" — works with both たら and ば and does not change the meaning, only adds emphasis.
- **Key Insight:** When in doubt, たら is the safer choice — it works in almost all situations. ば is more restricted and specifically implies a positive or wished outcome.
- **Cultural Note:** Japanese monkeys (さる) bathing in hot springs at Jigokudani Monkey Park is a famous winter sight in Japan — the lesson uses this as a cultural hook for the sentence できれば、さるを見に行きたいです。
