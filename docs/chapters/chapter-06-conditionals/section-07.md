## 2. Kanji Details

### 乗 — to get on; to ride

| | |
|-|-|
| **Reading** | の-る |
| **Words** | 乗る (のる) — to get on; to ride<br>乗りかえる (のりかえる) — to change / transfer (transport) |
| **Example** | バスに乗る時間を伝えます。*(Basu ni noru jikan wo tsutaemasu.)* — "I'll tell you the time to get on the bus." |

---

### 降 — to get off; to fall

| | |
|-|-|
| **Readings** | お-りる / ふ-る |
| **Words** | 降りる (おりる) — to get off<br>降る (ふる) — to fall (rain / snow) |
| **Radical** | β — meaning "hill" or "mound" |
| **Note** | 降 has two distinct readings: **おりる** (getting off transport) and **ふる** (precipitation falling). |
| **Examples** | 降りる - おりる、降る - ふる *(to get off, to fall)* |
| | 雨が降る。*(Ame ga furu.)* — "It rains." |

---

### 急 — to hurry

| | |
|-|-|
| **Reading** | いそ-ぐ |
| **Words** | 急ぐ (いそぐ) — to hurry |
| **Radical** | 心 (こころ) — meaning "heart", used for thoughts and emotions |
| **Same radical** | 思 (to think), 悲 (sad) |
| **て form** | 急いで (いそいで) — also used as an adverb: "hurriedly" |
| **Note** | 急ぐ is a U-verb. The て form 急(いそ)いで is used both as a request (急いでください) and as an adverb before another verb. |
| **Examples** | 急いでバスを乗りかえます。*(Isoide basu wo norikaemasu.)* — "I'll hurry and change buses." |
| | 雨が降るかもしれないので、急いで家に帰ります。*(Ame ga furu kamoshirenai no de, isoide ie ni kaerimasu.)* — "Because it might rain, I'll hurry and go home." |
| | 急ぎましたが、電車に間に合わなくて、遅れました。*(Isogimashita ga, densha ni maniawarakute, okuremashita.)* — "I hurried, but I didn't make it in time for the train and I was late." |

---

### 終 — to finish

| | |
|-|-|
| **Reading** | お-わる |
| **Words** | 終わる (おわる) — to finish |
| **Radical** | 糸 (いと) — meaning "thread" |
| **Same radical** | 紙（かみ）— paper |
| **Antonym** | 始める (はじめる) — to start / begin |
| **Examples** | 映画が終わりました。*(Eiga ga owarimashita.)* — "The film is finished." |

---

### 遅 — to be late

| | |
|-|-|
| **Reading** | おく-れる |
| **Words** | 遅れる (おくれる) — to be late |
| **Radical** | 辶 — meaning "to go forward" |
| **Same radical** | 道（みち）— street<br>運（はこぶ）— to carry |
| **Examples** | 飛行機に遅れる。*(Hikouki ni okurero.)* — "I'll be late for the plane." |
| | 急ぎましたが、電車に間に合わなくて、遅れました。*(Isogimashita ga, densha ni maniawarakute, okuremashita.)* — "I hurried, but I didn't make it in time for the train and I was late." |

## 4. Learning Context

- **Kanji Focus:** 乗・降・急・終・遅 — all transport and action-sequence related kanji.
- **Key Insight — 降:** Has two readings: おりる (get off transport) and ふる (rain/snow falling) — both share the sense of "coming down."
- **Key Insight — 急:** Radical 心 (heart) links it to emotion-driven urgency; て form 急いで doubles as an adverb meaning "hurriedly."
- **Key Insight — 終:** Radical 糸 (thread) — connect it to the idea of something "running out" or reaching its end.
- **Key Insight — 遅:** Radical 辶 (to go forward) — same as 道 and 運, all involving movement.
- **Context:** All five kanji appear in everyday transport and daily life sentences in Japan.
