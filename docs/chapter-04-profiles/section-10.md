## 2. Reading — The Marathon

**Japanese text (kanji):**

来週の日曜日、マラソン大会があります。やく42キロを走ります。今までにマラソンのけいけんがないので、とてもどきどきしています。友だちは楽しそうですが、私はちょっとしんぱいです。マラソンのコースは家の近くに作られたので、毎日そこを走っています。走り方を考えながられん習しています。走ることが大好きなので、がんばります。

**Japanese text (hiragana):**

らいしゅうのにちようび、マラソンたいかいがあります。やく42キロをはしります。いままでにマラソンのけいけんがないので、とてもどきどきしています。ともだちはたのしそうですが、わたしはちょっとしんぱいです。マラソンのコースはいえのちかくにつくられたので、まいにちそこをはしっています。はしりかたをかんがえながられんしゅうしています。はしることがだいすきなので、がんばります。

**English translation:**

There's a marathon next Sunday. We will run around 42 kilometres. I don't have any experience running marathons, so my heart is pounding nervously. My friend looks like she is having fun, but I am a bit worried. The marathon course has been set up near my house, so I run there every day. I think about how to run while I'm practising. I like running a lot, so I will do my best.

## 3. Grammar Points

### 〜なので — Because / Since

**Structure:** `[clause] + なので、[result]`

- マラソンのけいけんがないので、とてもどきどきしています。*(Marason no keiken ga nai no de, totemo dokidoki shite imasu.)* — I have no marathon experience, so my heart is pounding nervously.
- マラソンのコースは家の近くに作られたので、毎日そこを走っています。*(Marason no koosu wa ie no chikaku ni tsukurareta no de, mainichi soko wo hashitte imasu.)* — Because the marathon course was set up near my house, I run there every day.
- 走ることが大好きなので、がんばります。*(Hashiru koto ga daisuki na no de, ganbarimasu.)* — Because I love running, I will do my best.

### Passive in Natural Text

The passive 作られた (was built/set up) appears naturally mid-sentence as a subordinate clause — not isolated, but connected with なので.

- マラソンのコースは家の近くに作られた。*(Marason no koosu wa ie no chikaku ni tsukurareta.)* — The marathon course was set up near my house.

### 〜ことが大好き — Nominalisation + Preference

**Structure:** `Verb (dictionary) + ことが + 大好き／好き／きらい`

- 走ることが大好きなので、がんばります。*(Hashiru koto ga daisuki na no de, ganbarimasu.)* — Because I love running, I will do my best.

### ながら — Simultaneous Actions (in text)

- 走り方を考えながられん習しています。*(Hashirikata wo kangaenagara renshuu shite imasu.)* — I am practising while thinking about how to run.

## 5. Learning Context

- **なので vs から:** Both express reason/cause. なので is slightly more formal/written; から is more spoken/casual. In Yuki's text, なので appears three times — natural for a first-person written narrative.
- **どきどきしている:** どきどき is an onomatopoeia for a racing heartbeat. Paired with しんぱいです (worried), not さびしい — a comprehension trap in the exercises.
- **作られた in context:** The passive here means "was set up / was created" — the agent (who made the course) is omitted. This is the most common passive usage in Japanese: de-emphasising the agent.
- **ながら with れん習:** 走り方を考えながられん習している = practising while thinking about form. The ながら action (考える) is the background; れん習する is the main activity.
- **やく + number:** やく (approximately) always precedes the number — やく42キロ, やく3時間. Never follows.
- **走ること vs 走るの:** Both nominalisations work, but ことが大好き is the standard pattern for preference statements at B1 level.
