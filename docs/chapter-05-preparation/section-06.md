# Describing preparations done in advance

## 1. Grammar — て form ＋ ある (Resulting State)

This lesson introduces **て form ＋ ある**, which is used to describe the **resulting state** after something has been done on purpose.

### Contrast with て form ＋ おく

| Structure | Focus | Example |
|-----------|-------|---------|
| て form ＋ **おく** | The action done in advance | 私は飛行機のよやくをしておきました。*(I booked a flight in advance.)* |
| て form ＋ **ある** | The resulting state that remains | 飛行機のよやくがしてあります。*(A flight has been booked.)* |

### Key differences in particle usage

When switching from おく to ある, the **particle changes**:

| おく sentence | ある sentence |
|-------------|-------------|
| パスポートを かばんに 入れておきました。 | パスポート**が** かばんに 入れてあります。 |
| ひこうきのよやくをしておきます。 | ひこうきのよやく**が**してあります。 |

The object marker **を → が** when using てある.

## 3. Listening — Airport Check-in Dialogue

**Scene:** A ground crew member (グランドスタッフ) at the check-in desk is showing a boarding pass.

| Japanese | English |
|----------|---------|
| この飛行機のゲートは十時に開きます。 | The gate for this flight opens at ten o'clock. |
| ゲートの番号はここに書いてあります。 | The gate number has been written here. |
| ゲートの行き方はここに書いてあります。後で見てください。 | How to get to the gate has been written here. Please look at it later. |
| ざせきの番号はここに書いてあります。 | The seat number has been written here. |
| これは入国カードです。飛行機を降りる前に書いておいてください。いってらっしゃい！ | This is an arrival card. Please fill it in before getting off the plane. Have a safe trip! |

**Vocabulary from the dialogue:**

| Japanese | Reading | English |
|----------|---------|---------|
| グランドスタッフ | gurandosutaffu | Ground crew / Check-in staff |
| ゲート | geeto | Gate |

## 4. True / False — Quiz Answers

| Statement | Answer | Reason |
|-----------|--------|--------|
| ドアが開けてあります。The door is open (because someone opened it on purpose). | **True** | てある = resulting state from intentional action |
| Both the address and the phone number of the hotel have been written down. | **False** | 住所はここに書いてあります。でも、電話番号は分かりません。 — Only the address was written; she didn't know the phone number. |

## 5. Cultural Note — 入国カード (Arrival Card)

Everyone must submit a **入国カード** (arrival/landing card) at passport control when arriving in Japan.

- The card is available in multiple languages, but the kanji on it include:
  - 名（な）— name
  - 住所（じゅうしょ）— address
  - 国（くに）— country
  - 生年月日（せいねんがっぴ）— date of birth
- When asked to show your card, you can say: **書いてあります！** — "It has been filled in!"
- At some airports you can submit via app: **インストールしてあります！** — "It has been installed!"

## 6. Learning Context

- **Lesson:** Describing preparations done in advance
- **Chapter:** Chapter 5 — Preparation
- **Key grammar:** て form ＋ ある (resulting state after intentional action)
- **Particle rule:** を changes to が when switching from ておく to てある
- **Common mistake:** Using ておきます instead of てあります when describing the *current state* (not the past action)
  - とけいがなおし**てあります** ✓ (the watch has been fixed — current state)
  - とけいがなおし**ておきます** ✗ (wrong — this would mean "I'll fix the watch in advance")
